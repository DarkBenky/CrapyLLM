# Flask App

A simple Flask application.

## Setup and Run

1. **Create a Virtual Environment**  

   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows: .\venv\Scripts\activate
   ```

2. **Install Requirements**

    ```bash
    pip install -r requirements.txt
    ```

3. **Run the Application**

    ```bash
    python3 app.py
    ```

4. **Train Model**

    ```bash
    python3 model.py
    ```
